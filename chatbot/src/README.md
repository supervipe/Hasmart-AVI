# AVI_MOZ
Experimento de Avaliação de Serviços com Chatbot.

Fase Mágico de Oz.

## Antes de Treinar o Modelo
Baixar o modelo português do spaCy:

`python -m spacy download pt_core_news_lg`

## Treinar o Modelo
`rasa train`

## Ativar Actions
Em uma outra aba
`rasa run actions`

## Testar o Modelo
`rasa shell`
